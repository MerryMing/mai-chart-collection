# 抄谱流程

## 使用工具

Visual Mai: 太好用了你们知道吗? 鼠标点点就能抄谱,抄得又快又好!

Majdata: 核验谱面 Note 数量, 以及 Visual Mai 做不到的它来做

MaiChartConverter: 用于转换 maidata 到

MaiChartManager: 用于整理谱面, 方便编辑 XML 文件

Adobe Premiere: 用于编辑视频
